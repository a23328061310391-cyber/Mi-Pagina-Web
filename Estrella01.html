Ciberseguridad 01

Tipos de Malware

📚 Creado por: Estrella Lizeth
Conoce los diferentes tipos de software malicioso que pueden afectar la seguridad de tus sistemas y datos.
Tipo de Malware Función Impacto
VirusAuto replicable, infecta archivosDaño de datos al igual que de sistema
GusanosSe propaga a través de las redesInterrupción de la red, agotamiento de recursos
Caballo de TroyaSe disfraza de software legítimoRobo de datos, control de sistema
Software EspíaSupervisa en secreto la actividad del usuarioViolación de privacidad, robo de datos
Software PublicitarioMuestra anuncios no deseadosMolestia, degradación del rendimiento
RootkitOculta software maliciosoAcceso no autorizado, amenazas persistentes
KeyloggerRegistra las pulsaciones de teclasRobo de credenciales, filtración de datos
Fileless MalwareCorre en la memoriaAtaques sigilosos y difíciles de detectar
CryptojackerMinas de criptomonedasProblemas de rendimiento, secuestro de recursos
Red de Bots (Botnet)Crea una red de dispositivos infectadosAtaque DDoS, distribución de spam
Bomba LógicaEjecuta código en el disparadorDestrucción de datos, daños al sistema
WiperDestruye datosPérdida irreversible de datos
