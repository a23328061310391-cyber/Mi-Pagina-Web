Ciberseguridad 02

Tipos de Ataques Informáticos

📚 Creado por: Estrella Lizeth
Conoce los diferentes tipos de ataques que pueden comprometer la seguridad de sistemas y datos.
Tipo de Ataque Función / Cómo actúa Impacto
Phishing Engaña al usuario para que revele información sensible Robo de contraseñas, datos bancarios, identidad
Ransomware Secuestra archivos y pide rescate Pérdida de datos, paraliza sistemas
Ataque DDoS Satura servidores con tráfico masivo Interrupción de servicios, caída de sitios web
Inyección SQL Inserta comandos maliciosos en bases de datos Robo o destrucción de información
Cross-Site Scripting (XSS) Inserta scripts maliciosos en páginas web Robo de cookies, control de cuentas, redirecciones
Man-in-the-Middle (MitM) Intercepta comunicaciones entre dos partes Robo de información, manipulación de datos
Spoofing Suplantación de identidad o dirección IP Acceso no autorizado, engaños a usuarios
Malware móvil Software malicioso en smartphones Robo de información, control del dispositivo
Ingeniería social Manipulación psicológica para obtener datos o acceso Robo de información, acceso a sistemas críticos
Exploit Aprovecha vulnerabilidades del software Daño al sistema, control remoto, robo de datos
